# Github 블로그 구글 검색 실패

내 Github 블로그가 구글 검색이 가능하도록 구글링을 통해 방법을 따라했는데 생각대로 잘 안된다...
<br>

구글 서치 콘솔에서 html 인증 받고, `sitemap.xml` 파일을 생성하고 `robots.txt` 파일을 생성했다. 그리고 `sitemap.xml` 파일을 구글 서치 콘솔에 추가했는데 2일동안 가져올 수 없는 상태이다.
